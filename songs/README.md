## 🎨 創作理念

呢個 project 嘅目標係用現代音樂語言重新詮釋同創造：

- **保留** 經典嘅意象同核心情感
- **加入** 現代音樂元素（K-Pop、synth-pop、dark pop、trap、hyperpop⋯）
- **融合** 多語言（中文、英文、韓文）
- **實驗** 唔同曲風嘅可能性
- **致敬** (G)I-DLE、aespa、BLACKPINK 等 K-Pop 女團美學
