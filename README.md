### 🌙 [《錦瑟 · 夜》](./錦瑟-夜-lyrics.md) · [🎧 Listen](./audio/錦瑟-夜-yuqi.mp3)

> 原詩：李商隱《錦瑟》

**風格：** 溫柔古風 ballad  
**BPM：** 60-70（慢板）  
**適合：** 夜晚獨處、月下、靜思  
**配樂：** 古箏 + 鋼琴 + 弦樂  
**人聲：** 溫柔女聲，帶少少氣音  
**長度：** ~3 分鐘

**Hook 設計：**

- 「月光灑落無聲無言」做意境營造
- Outro 漸弱，營造夜晚嘅寧靜感

## 🎨 創作理念

唐詩本身就係古代嘅「歌」，有音韻、有節奏、有意境。

呢個 project 嘅目標係用現代音樂語言重新詮釋呢啲經典詩詞：

- **保留** 原詩嘅意象同核心情感
- **加入** 現代音樂元素（K-Pop、古風 ballad、Jazz、R&B⋯）
- **融合** 多語言（中文、英文、韓文）
- **實驗** 唔同曲風嘅可能性
